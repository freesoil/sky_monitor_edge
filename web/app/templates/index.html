🔌 API Endpoints

Device Configuration

POST /api/device/configure

Configure and save device IP address and port for persistent connection.

Device Status

GET /api/camera/status

Get real-time device status including camera, WiFi, SD card, and memory info.

Camera Settings

POST /api/camera/setting

Update individual camera parameters like framesize, quality, brightness, contrast.

Recording Config

POST /api/camera/recording-setting

Configure recording intervals, duration, and streaming parameters.

Device Commands

POST /api/camera/command

Send commands: start, stop, pause, restart, photo capture.

Camera Stream

GET /api/camera/stream

Get live camera image stream in JPEG format.
